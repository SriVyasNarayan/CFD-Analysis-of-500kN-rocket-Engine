# CFD-Analysis-of-500kN-rocket-Engine
Designed and Simulated a 500kN rocket engine which uses Hydrogen as fuel and Oxygen as oxidiser, using various softwares such as SOLIDWORKS and ANSYS Fluent.
Rocket Propulsion Analysis and NASA CEA were used to calculate the parameters defining the mass flow rate and dimensions of the model and a MATLAB code was written to calculate the parameters required.
 SOLIDWORKS was used to design the rocket engine and it was simulated in ANSYS Fluent.
 A correlation of 7% was observed between the calculated numerical and simulated results. Ansys accounts for losses while NASA CEA does not account for losses in frozen chemistry.
